# Style-guide

The Style-guide is a tool to make combinations about typography, colors, components.

`https://styleguide.michael.eti.br/`

## Stack:

- React;
- Redux;
- Redux-thunk;
- Duck pattern;
- Google Fonts API;
- styled-components
- Hooks.

## New features is on the way

- A backend was build for the new version using:
- Node.js;
- Express;
- MongoDB.
